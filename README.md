# concat-blobs

reads paths and configuration for concatinating blob scripts

#### config example
~~~~
{
  "filePaths": {
    "blob1": "./blobs/blob1.js",
    "blob2": "./blobs/blob2.js",
  },
  "cc": {
    "script1: {
      "feature1": "blob1"
    }
  }
}
~~~~ 

#### how to build the scripts
> npm start
